CSS Forms


Using HTML, we can create a simple form but that will not look attractive and beautiful.
Using CSS, we can apply styles on the HTML forms to make it colorful and attractive.





You can add Icons in a form in such a way that it might look inside the input text or outside the input text.
Either you can use your own icons or use icons like font awesome, google font icons, etc.
