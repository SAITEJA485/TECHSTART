Java Math


The Java Math class has many methods that allows you to perform mathematical tasks on numbers.

1. The Math.max(x,y) method can be used to find the highest value of x and y.

Example: 
			public class MaxExample1   
			{  
			    public static void main(String args[])  
			    {  
			        int x = 20;  
			        int y = 50;  
			        //print the maximum of two numbers   
			        System.out.println(Math.max(x, y));  
			    }  
			}

2. The Math.min(x,y) method can be used to find the lowest value of x and y.

Example: 
			public class MinExample1   
			{  
			    public static void main(String args[])  
			    {  
			        int x = 20;  
			        int y = 50;  
			        //print the maximum of two numbers   
			        System.out.println(Math.min(x, y));  
			    }  
			}

3. The Math.sqrt(x) method returns the square root of x.

Example: 
			public class SqrtExample1  
			{  
			    public static void main(String[] args)   
			    {  
			        double x = 81.0;  
			        // Input positive value, Output square root of x  
			        System.out.println(Math.sqrt(x));  
			    }  
			}

4. The Math.abs(x) method returns the absolute (positive) value of x.

Example: 
			public class AbsExample1  
			{  
			    public static void main(String args[])  
			    {  
			        int x = 78;  
			        int y = -48;  
			        //print the absolute value of int type  
			        System.out.println(Math.abs(x));  
			        System.out.println(Math.abs(y));  
			        System.out.println(Math.abs(Integer.MIN_VALUE));  
			    }  
			}

5. Math.random() returns a random number between 0.0 (inclusive), and 1.0 (exclusive).

Example: 
			public class RandomExample1  
			{  
			    public static void main(String[] args)   
			    {  
			        // generate random number  
			        double a = Math.random();  
			        double b = Math.random();  
			        // Output is different every time this code is executed    
			        System.out.println(a);  
			        System.out.println(b);  
			    }  
			}

6. int randomNum = (int)(Math.random() * 101); // 0 to 100.

Example: 
			class MyClass 
			{
			  int randomWithRange(int min, int max)
			  {   //defining method for a random number generator

			    int range = (max - min) + 1;     
			    return (int)(Math.random() * range) + min;
			  }  
			  public static void main( String args[] ) 
			  {
			    MyClass obj1=new MyClass();       // creating an object of MyClass
			    int rand=obj1.randomWithRange(1,100); // range is from 1 to 100 
			    System.out.println(rand);
			  }
			}
