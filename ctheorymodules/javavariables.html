Java Variables

A variable is a container which holds the value while the Java program is executed. A variable is assigned with a data type.

Variable is a name of memory location.

There are three types of variables in java: local, instance and static.

1.Local Variable
a.A variable declared inside the body of the method is called local variable. b.A local variable cannot be defined with "static" keyword.

2. Instance Variable
a.A variable declared inside the class but outside the body of the method, is called instance variable. b.It is not declared as static.

3.Static variable
a.A variable which is declared as static is called static variable. It cannot be local. b.You can create a single copy of static variable and share among all the instances of the class. c.Memory allocation for static variable happens only once when the class is loaded in the memory.
